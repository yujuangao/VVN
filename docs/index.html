Get Started

vvnthemes v0.0.1

Virginia Tech brand compliance for all Visualizing Virginia Numbers projects

devtools::install_github("yujuangao/VVN")
View on GitHub →

vvnthemes provides Virginia Tech brand colors, ggplot2 themes, map styling, styled tables, Shiny UI components, and project scaffolding for all VVN projects. It enforces consistent visual identity — VT Maroon, Burnt Orange, and Navy — across every chart, map, table, dashboard, and narrative report your team produces.

Features

  • VT brand colors and ggplot2 theme applied globally with a single call to set_vvn_defaults()
  • Discrete and continuous color palettes for charts and maps
  • Styled gt tables with maroon header, alternating rows, and source notes
  • Interactive Leaflet county choropleth maps for Shiny dashboards
  • Shiny UI components: filters, sliders, buttons, and KPI cards
  • One-command scaffolding for complete Shiny dashboards and Quarto narrative reports

Installation

To install vvnthemes, you need the devtools package:

install.packages("devtools")
devtools::install_github("yujuangao/VVN")

The source code and full release history are available at github.com/yujuangao/VVN.

Build with the library

Load vvnthemes at the top of every script or Quarto document. Call set_vvn_defaults() once per session — it applies VT Maroon and the VVN ggplot2 theme to all subsequent charts automatically. Every published figure requires vvn_source() for data attribution, and vvn_save() exports at 300 dpi.

## Build with the library

Load `vvnthemes` at the top of every script or Quarto document. Call `set_vvn_defaults()` once per session — it applies VT Maroon and the VVN ggplot2 theme to all subsequent charts automatically. Every published figure requires `vvn_source()` for data attribution, and `vvn_save()` exports at 300 dpi.

```{=html}
<div class="demo-panel demo-panel--stacked">
<div class="demo-panel-code">
<div class="demo-panel-header">R — county choropleth map</div>
<div class="demo-panel-body">
<pre><code id="demo-gs-code"></code><span class="demo-cursor" id="demo-gs-cursor"></span></pre>
</div>
</div>
<div class="demo-panel-output" id="demo-gs-output" style="opacity:0;transition:opacity 0.7s ease;">
<div class="demo-panel-header">Map output</div>
<div class="demo-panel-body">
<img src="images/demo-map.png" alt="VVN county choropleth map of Virginia">
</div>
</div>
</div>
<script>
(function(){
var code = [
'library(vvnthemes)',
'library(leaflet)',
'library(sf)',
'',
'# Load Virginia county boundaries',
'va <- vvnthemes::va_counties',
'',
'# Join your data (FIPS + value)',
'va$value <- c(0.42, 0.61, 0.55, 0.73, 0.38,',
'              0.67, 0.49, 0.81, 0.57, 0.44)',
'',
'# Build choropleth map',
'vvn_map_style(',
'  data        = va,',
'  variable    = "value",',
'  palette     = "maroon_seq",',
'  title       = "Economic Index by County, 2023",',
'  subtitle    = "Virginia ACS 5-Year Estimates",',
'  tooltip_var = "value",',
'  label_fmt   = scales::label_percent()',
')'
].join('\n');
function startDemo() {
  var el = document.getElementById('demo-gs-code');
  var cur = document.getElementById('demo-gs-cursor');
  var out = document.getElementById('demo-gs-output');
  if (!el) return;
  var i = 0, revealed = false;
  function type(){
    if(i < code.length){
      el.textContent += code[i++];
      if(!revealed && i >= Math.floor(code.length * 0.2)){
        revealed = true;
        if(out) out.style.opacity = '1';
      }
      setTimeout(type, i < 80 ? 35 : 16);
    } else {
      if(cur) cur.style.display = 'none';
      if(!revealed && out) out.style.opacity = '1';
    }
  }
  setTimeout(type, 700);
}
if(document.readyState === 'loading'){
  document.addEventListener('DOMContentLoaded', startDemo);
} else {
  startDemo();
}
})();
</script>
```

The same pattern applies to maps (`theme_vvn_map()`, `scale_fill_vvn_c()`) and tables (`gt()` |> `vvn_table()`). Step-by-step tutorials are available for [charts](charts.html), [maps](maps.html), [tables](tables.html), and [interactive Shiny components](interactive-components.html).
