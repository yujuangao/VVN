Get Started

vvnthemes v1.0.0

Virginia Tech brand compliance for all Visualizing Virginia Numbers projects

devtools::install_github("vt-vvn/vvnthemes")

vvnthemes provides Virginia Tech brand colors, ggplot2 themes, map styling, styled tables, Shiny UI components, and project scaffolding for all VVN projects. It enforces consistent visual identity — VT Maroon, Burnt Orange, and Navy — across every chart, map, table, dashboard, and narrative report your team produces.

Features

  • VT brand colors and ggplot2 theme applied globally with a single call to set_vvn_defaults()
  • Discrete and continuous color palettes for charts and maps
  • Styled gt tables with maroon header, alternating rows, and source notes
  • Interactive Leaflet county choropleth maps for Shiny dashboards
  • Shiny UI components: filters, sliders, buttons, and KPI cards
  • One-command scaffolding for complete Shiny dashboards and Quarto narrative reports

Installation

To install vvnthemes, you need the devtools package:

install.packages("devtools")
devtools::install_github("vt-vvn/vvnthemes")

Build with the library

Load vvnthemes at the top of every script or Quarto document. Call set_vvn_defaults() once per session — it applies VT Maroon and the VVN ggplot2 theme to all subsequent charts automatically. Every published figure requires vvn_source() for data attribution, and vvn_save() exports at 300 dpi.

R — horizontal bar chart
Chart output

The same pattern applies to maps (theme_vvn_map(), scale_fill_vvn_c()) and tables (gt() |> vvn_table()). Step-by-step tutorials are available for charts, maps, tables, and interactive Shiny components.

Use the Dashboard Template

create_vvn_dashboard() scaffolds a complete, deployable Shiny dashboard in one command. The dashboard runs immediately with placeholder data — no configuration required before the first preview:

library(vvnthemes)
create_vvn_dashboard("my-dashboard", title = "Virginia Housing Dashboard")
shiny::runApp("my-dashboard")

The dashboard opens in your browser with a sidebar, four KPI cards, a trend line chart, a horizontal bar chart, and a searchable data table — all VVN-styled. To populate it with your own data, open R/data_prep.R and replace the data source:

app_data <- readr::read_csv("data/my_data.csv")

Your data frame requires at minimum region (character), year (integer), and value (numeric). All charts, cards, and tables update automatically. See the Dashboard Template for a fully annotated walkthrough of all 11 sections.

Use the VVN Insights Template

create_vvn_story() scaffolds a complete Quarto narrative report with a branded hero section, written analysis, charts, a map, and a summary table:

library(vvnthemes)
create_vvn_story(
  "my-insight",
  title  = "Broadband Access in Rural Virginia",
  author = "VVN Research Team"
)
quarto::quarto_render("my-insight/index.qmd")

The report renders immediately with placeholder content — a hero banner, overview, background, findings, and conclusion sections, all styled and ready to fill in. Open index.qmd, replace every [placeholder] with your written content, and update the data loading line:

df <- readr::read_csv("data/my_data.csv")

See the VVN Insights Template for a fully annotated walkthrough of all 12 sections.

## Build with the library

Load `vvnthemes` at the top of every script or Quarto document. Call `set_vvn_defaults()` once per session — it applies VT Maroon and the VVN ggplot2 theme to all subsequent charts automatically. Every published figure requires `vvn_source()` for data attribution, and `vvn_save()` exports at 300 dpi.

```{=html}
<div class="demo-panel">
<div class="demo-panel-code">
<div class="demo-panel-header">R — horizontal bar chart</div>
<div class="demo-panel-body">
<pre><code id="demo-gs-code"></code><span class="demo-cursor" id="demo-gs-cursor"></span></pre>
</div>
</div>
<div class="demo-panel-output" id="demo-gs-output" style="display:none;opacity:0;transition:opacity 0.6s ease;">
<div class="demo-panel-header">Chart output</div>
<div class="demo-panel-body">
<img src="images/demo-chart.png" alt="VVN horizontal bar chart showing Economic Index by Region">
</div>
</div>
</div>
<script>
(function(){
var code = [
'library(vvnthemes)',
'library(ggplot2)',
'set_vvn_defaults()',
'',
'df <- data.frame(',
'  region = c("Rural", "Small City", "Suburban",',
'             "Urban Fringe", "Urban Core"),',
'  value  = c(46.2, 52.1, 57.4, 61.8, 65.3)',
')',
'',
'p <- ggplot(df, aes(x = value, y = reorder(region, value))) +',
'  geom_col(width = 0.65) +',
'  geom_text(aes(label = round(value, 1)), hjust = -0.2, size = 3.5) +',
'  scale_x_continuous(expand = expansion(mult = c(0, .15))) +',
'  labs(',
'    title    = "Economic Index by Region, 2023",',
'    subtitle = "Virginia regions ranked by index value",',
'    x = "Index value", y = NULL',
'  ) +',
'  vvn_source("Source: VVN Example Data") +',
'  remove_ticks()',
'',
'vvn_save(p, "figures/chart.png")'
].join('\n');
var el = document.getElementById('demo-gs-code');
var cur = document.getElementById('demo-gs-cursor');
var out = document.getElementById('demo-gs-output');
var i = 0;
function type(){
  if(i < code.length){ el.textContent += code[i++]; setTimeout(type, i < 60 ? 40 : 20); }
  else {
    cur.style.display = 'none';
    out.style.display = 'block';
    setTimeout(function(){ out.style.opacity = '1'; }, 30);
  }
}
setTimeout(type, 600);
})();
</script>
```

The same pattern applies to maps (`theme_vvn_map()`, `scale_fill_vvn_c()`) and tables (`gt()` |> `vvn_table()`). Step-by-step tutorials are available for [charts](charts.html), [maps](maps.html), [tables](tables.html), and [interactive Shiny components](interactive-components.html).
