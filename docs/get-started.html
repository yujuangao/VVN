Get Started

Install vvnthemes and build your first chart, table, and map.

What you will build

By the end of this page you will have a publication-ready bar chart, a styled data table, and a county-level choropleth map — all matching the VVN design standard. Then you will scaffold a complete dashboard or Insights document in one command.

Step 1 — Install

Install vvnthemes once per machine. You do not need to reinstall it every session.

install.packages("devtools")                       # skip if already installed
devtools::install_github("vt-vvn/vvnthemes")

Then add these two lines to the top of every script or Quarto chunk:

library(vvnthemes)
set_vvn_defaults()   # one call transforms every subsequent ggplot2 chart

set_vvn_defaults() does three things: it sets the global ggplot2 theme to theme_vvn(), changes all default geom colors to VT Maroon, and makes sure your charts look like VVN without any extra code.

Step 2 — Your first chart

Goal: A horizontal bar chart ranking Virginia regions by an index value — white background, VT Maroon bars, maroon title, source line at the bottom.

This is the most common VVN chart type. Copy, run, and you will see the result immediately.

library(vvnthemes)
library(ggplot2)

set_vvn_defaults()

df <- data.frame(
  region = c("Rural", "Small City", "Suburban", "Urban Fringe", "Urban Core"),
  value  = c(46.2, 52.1, 57.4, 61.8, 65.3)
)

ggplot(df, aes(x = value, y = reorder(region, value))) +
  geom_col(width = 0.65) +
  geom_text(aes(label = round(value, 1)), hjust = -0.2, size = 3.5) +
  scale_x_continuous(expand = expansion(mult = c(0, .15))) +
  labs(
    title    = "Economic Index by Region, 2023",
    subtitle = "Virginia regions ranked by index value",
    x = "Index value", y = NULL
  ) +
  vvn_source("VVN Example Data") +
  remove_ticks()

What happened automatically because of set_vvn_defaults():

  • Bars filled VT Maroon — no fill = argument needed
  • White plot background instead of ggplot2’s default gray
  • VT Maroon title and horizontal grid lines only
  • Professional font sizing and spacing

vvn_source() adds the required source line in gray below the chart. remove_ticks() removes axis ticks — standard for bar charts.

For more chart types (line, scatter, histogram, boxplot), see Chart Library →

Step 3 — Your first table

Goal: A summary table with a maroon header row, alternating gray body rows, title, and source line — the standard VVN table format.

The gt package is required. Install it once with install.packages("gt").

library(vvnthemes)
library(gt)

df <- data.frame(
  Region       = c("Urban Core", "Urban Fringe", "Suburban", "Small City", "Rural"),
  `2023 Value` = c(65.3, 61.8, 57.4, 52.1, 46.2),
  `Change`     = c("+4.1", "+3.9", "+3.5", "+3.2", "+3.0"),
  check.names  = FALSE
)

df |>
  gt() |>
  vvn_table(
    title       = "Economic Index by Region",
    subtitle    = "Virginia, 2023 estimates",
    source_note = "Source: VVN Example Data"
  )

vvn_table() applies the full VVN table style in one call: maroon header, alternating rows, orange bottom border on the header, and the source note.

For ranking tables, regression tables, and interactive DT tables, see Table Library →

Step 4 — Your first map

Goal: A county-level choropleth map of Virginia using the maroon sequential palette — light counties have low values, dark maroon counties have high values.

You need county boundary data. The fastest way is the tigris package:

install.packages("tigris")   # once
install.packages("sf")       # once
library(vvnthemes)
library(ggplot2)
library(sf)
library(tigris)
library(dplyr)

# Download Virginia county boundaries
va <- counties("VA", cb = TRUE, resolution = "20m") |>
  st_transform(crs = 5070)

# Join your data (replace with your own)
# va <- left_join(va, your_data, by = c("GEOID" = "fips"))

# For this example, simulate a value
set.seed(42)
va$rate <- runif(nrow(va), 30, 80)

ggplot(va, aes(fill = rate)) +
  geom_sf(color = "#FFFFFF", linewidth = 0.3) +
  scale_fill_vvn_c("maroon_seq") +
  theme_vvn_map() +
  labs(
    title    = "Broadband Access by County, 2023",
    subtitle = "Percentage of households with broadband access",
    fill     = "Households (%)"
  ) +
  vvn_source("FCC Broadband Data · VVN Analysis")

scale_fill_vvn_c("maroon_seq") applies the maroon sequential palette — the VVN standard for continuous county-level data. theme_vvn_map() removes axes and ticks automatically.

For interactive Leaflet maps and point maps, see Map Library →

Step 5 — Start a full project

Instead of building file by file, scaffold a complete VVN project in one command. The generated project runs immediately with placeholder data.

Dashboard (R Shiny):

library(vvnthemes)

create_vvn_dashboard("my-dashboard",
                      title = "Virginia Housing Dashboard")
# Created: my-dashboard/app.R · R/data_prep.R · www/vvn.css

shiny::runApp("my-dashboard")   # runs immediately

VVN Insights document (Quarto):

create_vvn_story("my-insight",
                  title  = "Broadband Access in Rural Virginia",
                  author = "VVN Research Team")
# Created: my-insight/index.qmd · _quarto.yml · styles.scss

quarto::quarto_render("my-insight/index.qmd")   # renders immediately

Replace the placeholder data and you have a complete VVN product.

See Dashboard Template → and VVN Insights Template → for annotated walkthroughs.
