VVN Insight

Scaffold and build a complete VVN narrative report.

The create_vvn_story() function scaffolds a complete Quarto narrative report in one command. The document renders immediately with placeholder content — you only replace the data and written sections.

Create the project

library(vvnthemes)
create_vvn_story(
  "my-insight",
  title  = "Broadband Access in Rural Virginia",
  author = "VVN Research Team"
)
quarto::quarto_render("my-insight/index.qmd")

Project structure

my-insight/
├── index.qmd       # Main document — edit this file
├── _quarto.yml     # Quarto configuration
├── styles.scss     # VVN brand styles (do not edit)
└── data/           # Place your data files here

What each file does

index.qmd is the complete narrative report. It contains all 12 sections — YAML front matter, setup chunk, hero banner, overview, background, data and methods, findings, discussion, conclusion, and rendering instructions — all as annotated placeholders ready to fill in.

_quarto.yml configures the Quarto project: theme, TOC settings, figure dimensions, and output format. You rarely need to edit this file.

styles.scss is the compiled VVN brand stylesheet. Do not edit this file.

data/ is an empty folder for your data files. Load them in the setup chunk with readr::read_csv("data/my_data.csv").

Section 8 — Findings: bar chart

Horizontal bar chart comparing the latest-year values across categories.

### Regional Comparison — Latest Year


::: {.cell}

```{.r .cell-code}
df |>
  filter(year == max(year)) |>
  mutate(region    = reorder(region, value),
         highlight = region == "Rural") |>
  ggplot(aes(x = value, y = region, fill = highlight)) +
  geom_col(width = 0.65) +
  geom_text(aes(label = round(value, 1)), hjust = -0.2, size = 3.5,
            color = "#3D3D3D") +
  scale_fill_manual(values = c(`TRUE` = "#E5751F", `FALSE` = "#861F41"),
                    guide  = "none") +
  scale_x_continuous(expand = expansion(mult = c(0, .15))) +
  labs(title    = paste("Regional Comparison ·", max(df$year)),
       subtitle = "Rural highlighted in orange",
       x = "Value", y = NULL,
       caption  = "Source: [Dataset name] · VVN Analysis") +
  remove_ticks()
```

Section 10 — Summary table

Summary statistics table using gt + vvn_table().

## Summary Table


::: {#tbl-summary .cell tbl-cap='Summary statistics by region, 2018–2023.'}

```{.r .cell-code}
summary_df <- df |>
  group_by(Region = region) |>
  summarise(
    `Mean`          = round(mean(value), 1),
    `2023 Value`    = round(value[year == max(year)], 1),
    `Change (pts)`  = round(value[year == max(year)] - value[year == min(year)], 1),
    .groups = "drop"
  ) |>
  arrange(desc(`2023 Value`))

library(gt)
gt(summary_df) |>
  vvn_table(title       = "Regional Summary Statistics, 2018–2023",
             source_note = "Source: [Dataset] · VVN Analysis")
```

## Publishing checklist

Before submitting a VVN Insight for publication:

- [ ] All `[...]` placeholders replaced
- [ ] Hero banner present with title and VVN subtitle
- [ ] Key finding stated clearly in Overview
- [ ] Background explains context and audience
- [ ] Data & Methods table complete with source
- [ ] At least 2 charts with titles, subtitles, and source lines
- [ ] At least 1 map (interactive or static)
- [ ] Summary table with source note
- [ ] Discussion covers interpretation and limitations
- [ ] Conclusion with contact information
- [ ] All chart titles use headline case
- [ ] All chart subtitles use sentence case
- [ ] `embed-resources: true` in YAML (single-file HTML)
