Charts

Create VVN-styled ggplot2 charts step by step.

For title/label conventions and chart selection rules, see Chart Standards →

Create your first chart

The horizontal bar chart is the most common VVN output. Follow these steps to build one from scratch.

Step 1 — Load packages

library(vvnthemes)
library(ggplot2)
set_vvn_defaults()

Call set_vvn_defaults() once per session. It sets the VVN theme globally and makes all bars, lines, and points VT Maroon by default — you never have to specify colors manually.

Step 2 — Prepare your data

df <- data.frame(
  region = c("Rural", "Small City", "Suburban", "Urban Fringe", "Urban Core"),
  value  = c(46.2, 52.1, 57.4, 61.8, 65.3)
)

Sort rows from smallest to largest value before plotting so bars display in order.

Step 3 — Build the chart

p <- ggplot(df, aes(x = value, y = reorder(region, value))) +
  geom_col(width = 0.65) +
  geom_text(aes(label = round(value, 1)), hjust = -0.2, size = 3.5) +
  scale_x_continuous(expand = expansion(mult = c(0, .15)))
  • x = value, y = reorder(region, value) — value on x, categories on y, sorted
  • geom_text(...) — adds a value label just past the end of each bar
  • scale_x_continuous(expand = ...) — adds space to the right so labels aren’t cut off

Step 4 — Add title, subtitle, and source

p <- p +
  labs(
    title    = "Economic Index by Region, 2023",
    subtitle = "Virginia regions ranked by index value",
    x = "Index value", y = NULL
  ) +
  vvn_source("VVN Example Data") +
  remove_ticks()
  • title — headline case: capitalize every major word
  • subtitle — sentence case: geography and year
  • vvn_source() — required on every published chart; adds a gray caption at the bottom
  • remove_ticks() — standard for bar charts; removes the small tick marks on axes

Step 5 — Save

vvn_save(p, "figures/chart.png")              # 8×5 in, 300 dpi
vvn_save(p, "figures/chart.png", width = 12)  # wider for wide layouts
vvn_save(p, "figures/chart", formats = c("png", "pdf"))  # both formats

Which chart should I make?

I want to show… Chart type Key function
Ranking across 5+ categories Horizontal bar geom_col() (bars go left-right)
Comparison of 3–4 short-named groups Vertical column geom_col() (bars go up)
Change over time Line chart geom_line() + geom_point()
Distribution of one variable Histogram geom_histogram()
Distributions across groups Boxplot geom_boxplot()
Relationship between two variables Scatter plot geom_point()

More chart types

Line chart

Use when Showing change over time for one or more groups. One line per group. 
df_trend <- data.frame(
  year  = rep(2018:2023, 2),
  value = c(44, 46, 47, 49, 51, 53, 62, 63, 65, 64, 67, 70),
  group = rep(c("Rural", "Urban"), each = 6)
)

ggplot(df_trend, aes(x = year, y = value, color = group, group = group)) +
  geom_line(linewidth = 1.2) +
  geom_point(size = 2.5) +
  scale_color_vvn("brand") +
  scale_x_continuous(breaks = 2018:2023) +
  labs(
    title    = "Broadband Access Over Time",
    subtitle = "Percentage of households, Virginia 2018–2023",
    x = NULL, y = "Households (%)", color = NULL
  ) +
  vvn_source("FCC Broadband Data") +
  legend_bottom()
Tip

For direct labels instead of a legend, use ggrepel::geom_text_repel() on the last year’s values and add remove_legend().

Scatter plot

Use when Exploring the association between two continuous variables. Always add grid lines on both axes. 
ggplot(df, aes(x = income, y = broadband)) +
  geom_point(size = 3, alpha = 0.8) +
  geom_smooth(method = "lm", se = FALSE, linewidth = 0.8) +
  labs(
    title    = "Income and Broadband Access",
    subtitle = "Virginia counties, 2022",
    x = "Median household income", y = "Households with broadband (%)"
  ) +
  vvn_source("U.S. Census Bureau, ACS 5-Year Estimates, 2022") +
  scatter_grid()

scatter_grid() adds grid lines on both axes — required for scatter plots so readers can trace values.

Histogram

Use when Showing the distribution of a single continuous variable. Add a mean line for context. 
m <- mean(df$value, na.rm = TRUE)

ggplot(df, aes(x = value)) +
  geom_histogram(bins = 25, fill = "#861F41", color = "#FFFFFF", alpha = 0.9) +
  geom_vline(xintercept = m, color = "#E5751F", linewidth = 1.2, linetype = "dashed") +
  annotate("text", x = m + 0.5, y = Inf,
           label = paste0("Mean: ", round(m, 1)),
           color = "#E5751F", hjust = 0, vjust = 1.5, size = 3.5) +
  labs(title = "Distribution of Values", x = "Value", y = "Count") +
  vvn_source("Source: [Dataset] · VVN Analysis")

Boxplot

Use when Comparing distributions across two or more groups. Shows median, IQR, and outliers. 
ggplot(df, aes(x = region, y = value, fill = region)) +
  geom_boxplot(width = 0.5, outlier.color = "#333333",
               outlier.size = 1.5, show.legend = FALSE) +
  scale_fill_vvn("main") +
  labs(title = "Value Distribution by Region", x = NULL, y = "Value") +
  vvn_source("Source: [Dataset] · VVN Analysis") +
  remove_ticks()

Highlight one group

Use when Drawing attention to a focal category — for example, Rural vs. all others. 
df_latest <- df |>
  filter(year == max(year)) |>
  arrange(value) |>
  mutate(region = factor(region, levels = region),
         highlight = region == "Rural")

ggplot(df_latest, aes(x = value, y = region, fill = highlight)) +
  geom_col(width = 0.65) +
  scale_fill_manual(values = c(`TRUE` = "#E5751F", `FALSE` = "#861F41"),
                    guide  = "none") +
  labs(title    = "Regional Comparison, 2023",
       subtitle = "Rural highlighted in orange",
       x = NULL, y = NULL) +
  vvn_source("Source: [Dataset] · VVN Analysis") +
  remove_ticks()

Color palettes

Palette Use for
scale_fill_vvn("main") General purpose — up to 10 groups
scale_fill_vvn("brand") 2–3 groups — maroon, orange, navy
scale_fill_vvn("accessible") Colorblind-safe alternative
scale_fill_vvn_c("maroon_seq") Continuous gradient — maps and heatmaps
scale_fill_vvn_c("diverging") Above / below average

The same palettes work for color (lines, points): scale_color_vvn("main").

Common mistakes

Bars aren’t maroon. You forgot set_vvn_defaults(). Add it once at the top of your script.

Value labels get cut off. Add scale_x_continuous(expand = expansion(mult = c(0, .15))) to create space to the right.

Bars aren’t sorted. Use reorder(region, value) in the aes() mapping, or arrange() + mutate(region = factor(region, levels = region)) before plotting.

Chart fails VVN review. You forgot vvn_source(). It is required on every published figure.

Scatter plot looks empty. You need scatter_grid() or theme_vvn(grid = "both") — scatter plots require grid lines on both axes.

## Create your first chart

The horizontal bar chart is the most common VVN output. Follow these steps to build one from scratch.

### Step 1 — Load packages

```r

```

```r
df <- data.frame(

```

Sort rows from smallest to largest value before plotting so bars display in order.


- `geom_text(...)` — adds a value label just past the end of each bar
- `scale_x_continuous(expand = ...)` — adds space to the right so labels aren't cut off

## Which chart should I make?

| I want to show… | Chart type | Key function |
|---|---|---|
| Ranking across 5+ categories | Horizontal bar | `geom_col()` (bars go left-right) |
| Comparison of 3–4 short-named groups | Vertical column | `geom_col()` (bars go up) |
| Change over time | Line chart | `geom_line()` + `geom_point()` |
| Distribution of one variable | Histogram | `geom_histogram()` |
| Distributions across groups | Boxplot | `geom_boxplot()` |
| Relationship between two variables | Scatter plot | `geom_point()` |

## Common mistakes

**Bars aren't maroon.** You forgot `set_vvn_defaults()`. Add it once at the top of your script.

**Value labels get cut off.** Add `scale_x_continuous(expand = expansion(mult = c(0, .15)))` to create space to the right.

**Bars aren't sorted.** Use `reorder(region, value)` in the `aes()` mapping, or `arrange()` + `mutate(region = factor(region, levels = region))` before plotting.

**Chart fails VVN review.** You forgot `vvn_source()`. It is required on every published figure.

**Scatter plot looks empty.** You need `scatter_grid()` or `theme_vvn(grid = "both")` — scatter plots require grid lines on both axes.
