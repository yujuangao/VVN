vvnthemes

Function Reference · Complete documentation for all vvnthemes exports.

Overview

vvnthemes is the design foundation for all VVN projects. It provides ggplot2 themes, brand color palettes, scale functions, Shiny UI components, and project scaffolding — all consistent with the Virginia Tech visual identity.

# Install once from GitHub
devtools::install_github("vt-vvn/vvnthemes")

# Load and apply
library(vvnthemes)
set_vvn_defaults()   # styles all subsequent ggplot2 calls

See Get Started → for a step-by-step workflow guide.

Setup

set_vvn_defaults()

Set theme_vvn() as the global ggplot2 theme and update default geom aesthetics for the session. Call once at the top of every script.

Usage

set_vvn_defaults(style = c("standard", "map"), base_size = 12)

Arguments

Argument Default Description
style "standard" "standard" applies theme_vvn(); "map" applies theme_vvn_map().
base_size 12 Base font size passed to the theme.

Value

Invisibly NULL. Called for its side effects — sets the ggplot2 global theme and geom color defaults (bars, lines, points → VT Maroon; smooth → VT Orange).

Examples

library(vvnthemes)
set_vvn_defaults()

# All subsequent ggplot2 calls use theme_vvn() and VVN colors automatically
ggplot2::ggplot(mpg, ggplot2::aes(displ, hwy)) +
  ggplot2::geom_point()

undo_vvn_defaults()   # revert when done

undo_vvn_defaults()

Revert to ggplot2 factory defaults, undoing set_vvn_defaults().

Usage

undo_vvn_defaults()

Value

Invisibly NULL.

Examples

set_vvn_defaults()
# ... make VVN charts ...
undo_vvn_defaults()

Themes

theme_vvn()

Standard VVN ggplot2 theme: white background, VT Maroon titles, light gray grid, and clean typography. Use for bars, lines, histograms, and most charts.

Usage

theme_vvn(base_size = 12, base_family = "sans", grid = "y")

Arguments

Argument Default Description
base_size 12 Base font size in points.
base_family "sans" Font family.
grid "y" Grid lines to show: "y" (horizontal only), "x", "both", or "none".

Value

A ggplot2::theme object.

Examples

library(ggplot2)

# Bar chart — horizontal grid (default)
ggplot(mpg, aes(class)) +
  geom_bar() +
  theme_vvn()

# Scatter plot — both axes
ggplot(mpg, aes(displ, hwy)) +
  geom_point() +
  theme_vvn(grid = "both")

theme_vvn_map()

Map theme for ggplot2 + sf static choropleth maps. Removes axes and ticks; adds a muted panel background and a horizontal legend at the bottom.

Usage

theme_vvn_map(base_size = 11, base_family = "sans")

Arguments

Argument Default Description
base_size 11 Base font size in points.
base_family "sans" Font family.

Value

A ggplot2::theme object.

Examples

library(ggplot2); library(sf)
ggplot(va_counties_sf, aes(fill = median_income)) +
  geom_sf() +
  scale_fill_vvn_c("maroon_seq") +
  theme_vvn_map()

theme_vvn_minimal()

Minimal VVN variant: no grid lines, no axes. Use for small multiples, annotation-heavy figures, and inset charts.

Usage

theme_vvn_minimal(base_size = 12, base_family = "sans")

Arguments

Argument Default Description
base_size 12 Base font size in points.
base_family "sans" Font family.

Value

A ggplot2::theme object.

Color System

vvn_colors()

Return one or more named VVN brand colors as a named character vector of hex codes.

Usage

vvn_colors(...)

Arguments

Argument Description
... Color name(s) to return. Valid names: "maroon", "orange", "navy", "charcoal", "gray", "light_gray", "white", "maroon_dk", "maroon_lt", "orange_lt". If none provided, returns all colors.

Value

A named character vector of hex codes.

Examples

vvn_colors()                             # all brand colors
vvn_colors("maroon")                     # "#861F41"
vvn_colors("maroon", "orange", "navy")   # three primary VVN colors

vvn_palette()

Generate a vector of hex colors from a named VVN palette. Sequential and diverging palettes are interpolated when n exceeds the base palette size.

Usage

vvn_palette(palette = "main", n = NULL, reverse = FALSE, alpha = 1)

Arguments

Argument Default Description
palette "main" Palette name. Categorical: "main" (10 colors), "brand" (3 colors: maroon/orange/navy), "accessible" (7 colors, Okabe-Ito). Sequential: "maroon_seq", "orange_seq", "navy_seq". Diverging: "diverging". Neutral: "gray_seq".
n NULL Number of colors to return. NULL returns all base palette colors.
reverse FALSE Reverse the palette order?
alpha 1 Opacity, 0–1.

Value

A character vector of hex color codes.

Examples

vvn_palette()                        # 10 main colors
vvn_palette("brand")                 # "#861F41" "#E5751F" "#1B5299"
vvn_palette("maroon_seq", n = 9)     # 9-step interpolated maroon
vvn_palette("diverging", n = 11)
vvn_palette("accessible")

view_vvn_palette()

Display a horizontal color swatch strip for any VVN palette.

Usage

view_vvn_palette(palette = "main", n = NULL, label = TRUE, title = NULL)

Arguments

Argument Default Description
palette "main" Palette name (see vvn_palette()).
n NULL Number of colors to display. NULL shows all base colors.
label TRUE Show hex codes on each swatch?
title NULL Plot title. Defaults to vvn_palette("<palette>").

Value

A ggplot object, printed invisibly. Save with vvn_save().

Examples

view_vvn_palette()                       # "main" palette
view_vvn_palette("diverging", n = 9)
view_vvn_palette("accessible", label = FALSE)

Discrete Scales

scale_color_vvn() / scale_fill_vvn()

Apply a VVN categorical palette to ggplot2 color or fill aesthetics. scale_colour_vvn() is a British-English alias for scale_color_vvn().

Usage

scale_color_vvn(palette = "main", reverse = FALSE, alpha = 1, ...)
scale_colour_vvn(palette = "main", reverse = FALSE, alpha = 1, ...)
scale_fill_vvn(palette = "main", reverse = FALSE, alpha = 1, ...)

Arguments

Argument Default Description
palette "main" Palette name: "main" (general purpose, up to 10 categories), "brand" (3 groups: maroon/orange/navy), or "accessible" (colorblind-safe).
reverse FALSE Reverse palette order?
alpha 1 Opacity, 0–1.
... Passed to ggplot2::discrete_scale().

Value

A ggplot2 scale object.

Examples

library(ggplot2)

# 3-group bar chart
ggplot(mpg, aes(class, fill = drv)) +
  geom_bar(position = "dodge") +
  scale_fill_vvn("brand")

# Multi-series line chart
ggplot(economics_long, aes(date, value01, color = variable)) +
  geom_line() +
  scale_color_vvn("main")

Continuous Scales

scale_color_vvn_c() / scale_fill_vvn_c()

Apply a VVN sequential or diverging palette to continuous ggplot2 color or fill aesthetics. scale_colour_vvn_c() is a British-English alias for scale_color_vvn_c().

Usage

scale_color_vvn_c(palette = "maroon_seq", reverse = FALSE, ...)
scale_colour_vvn_c(palette = "maroon_seq", reverse = FALSE, ...)
scale_fill_vvn_c(palette = "maroon_seq", reverse = FALSE, ...)

Arguments

Argument Default Description
palette "maroon_seq" Palette name: "maroon_seq" (primary), "orange_seq", "navy_seq", "diverging" (for above/below-average data), or "gray_seq".
reverse FALSE Reverse palette?
... Passed to ggplot2::scale_color_gradientn() or scale_fill_gradientn().

Value

A ggplot2 scale object.

Examples

library(ggplot2)

# Choropleth fill — most common VVN use
ggplot(va_counties_sf, aes(fill = pct_broadband)) +
  geom_sf() +
  scale_fill_vvn_c("maroon_seq") +
  theme_vvn_map()

# Diverging scale (z-scores, above/below average)
ggplot(df, aes(x, y, fill = z_score)) +
  geom_tile() +
  scale_fill_vvn_c("diverging")

Chart Helpers

vvn_title()

Add a formatted title and optional subtitle to a ggplot.

Usage

vvn_title(title, subtitle = NULL)

Arguments

Argument Default Description
title Title string. Use headline case (capitalize all major words).
subtitle NULL Optional subtitle. Use sentence case.

Value

A ggplot2::labs() call.

Examples

ggplot(mpg, aes(displ, hwy)) +
  geom_point() +
  vvn_title("Engine Size vs Fuel Economy",
             subtitle = "Motor Trend Car Road Tests, 1974")

vvn_source()

Add a source attribution caption to any ggplot. Required on every published VVN figure.

Usage

vvn_source(source, note = NULL)

Arguments

Argument Default Description
source Source string, e.g. "U.S. Census Bureau, ACS 5-Year Estimates, 2022".
note NULL Optional note prepended as "Note: ..." before the source line.

Value

A ggplot2::labs(caption = ...) call.

Examples

ggplot(df, aes(x, y)) +
  geom_point() +
  vvn_source("U.S. Census Bureau, ACS 5-Year Estimates, 2022")

# With a methodological note
ggplot(df, aes(x, y)) +
  geom_col() +
  vvn_source("Virginia Employment Commission",
              note = "Data suppressed for localities with fewer than 100 workers.")

vvn_note()

Add a footnote-style annotation below a ggplot.

Usage

vvn_note(text)

Arguments

Argument Description
text Note text. Prefixed with "Note: " in the caption.

Value

A ggplot2::labs(caption = ...) call.

Examples

ggplot(df, aes(x, y)) +
  geom_col() +
  vvn_note("Excludes localities with suppressed data.")

vvn_save()

Save a ggplot at publication quality with VVN default dimensions (8×5 in, 300 dpi, white background). Supports multi-format export in one call.

Usage

vvn_save(plot = ggplot2::last_plot(), filename, width = 8, height = 5,
         dpi = 300, formats = NULL)

Arguments

Argument Default Description
plot last_plot() A ggplot object. Defaults to the most recently printed plot.
filename File path with extension (e.g. "figures/chart.png"), or base path when formats is provided.
width 8 Width in inches.
height 5 Height in inches.
dpi 300 Resolution in dots per inch.
formats NULL Character vector of extensions for simultaneous multi-format export, e.g. c("png", "pdf", "svg"). Overrides the filename extension.

Value

Invisibly returns the file path(s).

Examples

p <- ggplot(mpg, aes(class)) + geom_bar() + theme_vvn()

vvn_save(p, "figures/class_bar.png")

# Export to multiple formats at once
vvn_save(p, "figures/class_bar", formats = c("png", "pdf", "svg"))

Tables

vvn_table()

Apply VVN branding to a gt table: VT Maroon column headers, alternating light gray rows, orange accent border, and optional title and source note.

Requires the gt package (install.packages("gt")).

Usage

vvn_table(gt_tbl, title = NULL, subtitle = NULL, source_note = NULL,
          striped = TRUE, compact = FALSE)

Arguments

Argument Default Description
gt_tbl A gt_tbl object from gt::gt().
title NULL Table title string. Use headline case.
subtitle NULL Optional subtitle. Use sentence case.
source_note NULL Source attribution string. Required for all published tables.
striped TRUE Apply alternating row shading (#F7F7F7)?
compact FALSE Reduce cell padding for dense tables?

Value

A styled gt_tbl object.

Examples

library(gt)

gt(head(mtcars)) |>
  vvn_table(
    title       = "Motor Trend Road Tests",
    subtitle    = "Selected vehicles, 1974",
    source_note = "Source: Henderson & Velleman (1981)"
  )

For extended examples including ranking tables, regression tables, and interactive DT tables, see the Table Library →.

vvn_table_pvalues()

Color-code p-values in a gt regression table: maroon for p < 0.001, orange for p < 0.05.

Usage

vvn_table_pvalues(gt_tbl, col_name = "p.value",
                  thresholds = c("0.001" = "#861F41", "0.05" = "#E5751F"))

Arguments

Argument Default Description
gt_tbl A gt_tbl object, typically piped after vvn_table().
col_name "p.value" Column name containing p-values (character).
thresholds c("0.001" = "#861F41", "0.05" = "#E5751F") Named vector mapping threshold values to highlight hex colors.

Value

A gt_tbl with highlighted p-value cells.

Examples

library(gt); library(broom)

lm(mpg ~ wt + cyl, data = mtcars) |>
  tidy() |>
  gt() |>
  vvn_table(title = "OLS Regression: Fuel Economy") |>
  vvn_table_pvalues()

Maps

vvn_map_style()

Apply VVN branding to a Leaflet choropleth map: muted CartoDB basemap, VVN palette fill, maroon county hover highlight, popup tooltips, and a styled legend.

Requires the leaflet and htmltools packages.

Usage

vvn_map_style(map, data, value_col, label_col = value_col,
              palette = "maroon_seq", n_bins = 6, title = value_col,
              opacity = 0.75,
              fmt_fn = function(x) format(round(x), big.mark = ","))

Arguments

Argument Default Description
map A leaflet map from leaflet::leaflet().
data An sf data frame with polygon geometry.
value_col Name of the numeric column to visualize (character).
label_col value_col Column used as the tooltip label (e.g. county name).
palette "maroon_seq" VVN palette: "maroon_seq", "orange_seq", "navy_seq", or "diverging".
n_bins 6 Number of legend bins.
title value_col Legend title string.
opacity 0.75 Fill opacity, 0–1.
fmt_fn comma format Function to format tooltip values. Default: format(round(x), big.mark = ",").

Value

A styled leaflet map.

Examples

library(leaflet); library(sf)

leaflet(va_counties) |>
  vvn_map_style(va_counties, "pct_broadband",
                label_col = "name",
                title     = "Broadband coverage (%)",
                palette   = "maroon_seq")

For complete map examples including variable selectors and diverging palettes, see the Map Library →.

vvn_accessibility_check()

Compute WCAG 2.1 contrast ratios for a set of hex colors against a background and report pass/fail status.

Usage

vvn_accessibility_check(colors, background = "#FFFFFF", level = c("AA", "AAA"))

Arguments

Argument Default Description
colors Character vector of hex codes to test.
background "#FFFFFF" Background hex code to test against.
level "AA" WCAG standard: "AA" (≥ 4.5:1 contrast ratio) or "AAA" (≥ 7.0:1).

Value

Invisibly returns a data frame with columns color, contrast_ratio, pass. Prints a formatted pass/fail report via cli.

Examples

# Check the brand palette on white
vvn_accessibility_check(vvn_palette("brand"))

# Check white text on a maroon background
vvn_accessibility_check(c("#FFFFFF", "#F7F7F7"), background = "#861F41")

# Stricter AAA standard
vvn_accessibility_check(vvn_palette("accessible"), level = "AAA")

## Overview

**vvnthemes** is the design foundation for all VVN projects. It provides ggplot2 themes, brand color palettes, scale functions, Shiny UI components, and project scaffolding — all consistent with the Virginia Tech visual identity.

```r
# Install once from GitHub
devtools::install_github("vt-vvn/vvnthemes")

# Load and apply
library(vvnthemes)
set_vvn_defaults()   # styles all subsequent ggplot2 calls
```

> See [Get Started →](get-started.html) for a step-by-step workflow guide.

---

## Setup

### `set_vvn_defaults()` {#set_vvn_defaults}

Set `theme_vvn()` as the global ggplot2 theme and update default geom aesthetics for the session. Call once at the top of every script.

#### Usage

```r
set_vvn_defaults(style = c("standard", "map"), base_size = 12)
```

#### Arguments

| Argument | Default | Description |
|----------|---------|-------------|
| `style` | `"standard"` | `"standard"` applies `theme_vvn()`; `"map"` applies `theme_vvn_map()`. |
| `base_size` | `12` | Base font size passed to the theme. |

#### Value

Invisibly `NULL`. Called for its side effects — sets the ggplot2 global theme and geom color defaults (bars, lines, points → VT Maroon; smooth → VT Orange).

#### Examples

```r
library(vvnthemes)
set_vvn_defaults()

# All subsequent ggplot2 calls use theme_vvn() and VVN colors automatically
ggplot2::ggplot(mpg, ggplot2::aes(displ, hwy)) +
  ggplot2::geom_point()

undo_vvn_defaults()   # revert when done
```

---

### `undo_vvn_defaults()` {#undo_vvn_defaults}

Revert to ggplot2 factory defaults, undoing `set_vvn_defaults()`.

#### Usage

```r
undo_vvn_defaults()
```

#### Value

Invisibly `NULL`.

#### Examples

```r
set_vvn_defaults()
# ... make VVN charts ...
```

### `create_vvn_dashboard()` {#create_vvn_dashboard}

Scaffold a production-ready R Shiny dashboard with VVN branding: maroon navbar, KPI cards, reactive charts, Leaflet county map, sidebar filters, and a data download tab.

#### Usage

```r
create_vvn_dashboard(name, path = ".", title = name,
                     author = "Visualizing Virginia Numbers")
```

#### Arguments

| Argument | Default | Description |
|----------|---------|-------------|
| `name` | | Project folder name. |
| `path` | `"."` | Parent directory. |
| `title` | `name` | Dashboard title shown in the maroon navbar. |
| `author` | `"Visualizing Virginia Numbers"` | Author name(s). |

#### Value

Invisibly returns the project path. Prints setup instructions via `cli`.

#### Details

Creates:

```
name/
├── app.R            ← Main dashboard (runs immediately with example data)
├── R/data_prep.R    ← Load and clean your data
├── www/vvn.css      ← VVN brand CSS
└── data/
    ├── raw/
    └── processed/
```

#### Examples

```r
create_vvn_dashboard("housing_dashboard",
                      title = "Virginia Housing Affordability Dashboard")

shiny::runApp("housing_dashboard")
rsconnect::deployApp("housing_dashboard")   # publish
```
