Function Index

Package {vvnthemes}

Quick-reference for every function in vvnthemes. Click any function name to jump to its full documentation in the Reference Manual.

Setup

Function Purpose
set_vvn_defaults() Sets theme_vvn() as the global ggplot2 theme and changes default geom colors to VT Maroon (bars, points, lines) and VT Orange (smooth lines). Call once per script — every chart after this is automatically branded.
undo_vvn_defaults() Reverts ggplot2 to factory defaults. Use at the end of a script or when switching to non-VVN work.

Themes

Function Purpose
theme_vvn() Standard VVN ggplot2 theme — white background, maroon titles, light gray horizontal gridlines. Use for bar charts, line charts, and histograms.
theme_vvn_map() Map variant for ggplot2 + sf static choropleth maps. Removes axes, adds muted panel background, moves legend to bottom.
theme_vvn_minimal() Stripped-down variant — no gridlines, no axes. For small multiples, annotation-heavy figures, or inset charts.

Color System

Function Purpose
vvn_colors() Returns hex codes for named VVN brand colors (maroon, orange, navy, charcoal, etc.). Use when you need a specific color value directly.
vvn_palette() Returns a vector of hex codes from a named palette ("main", "brand", "maroon_seq", "diverging", etc.). Sequential palettes interpolate to any number of steps.
view_vvn_palette() Renders a horizontal color swatch strip for any palette — use to preview available colors before building a chart.

Scales

Function Purpose
scale_color_vvn() Applies a VVN categorical palette to the color aesthetic. Use for grouped line charts and scatter plots.
scale_fill_vvn() Applies a VVN categorical palette to the fill aesthetic. Use for grouped bar charts and area charts.
scale_color_vvn_c() Applies a sequential or diverging palette to a continuous color aesthetic. Use for scatter plots where color encodes a numeric variable.
scale_fill_vvn_c() Applies a sequential or diverging palette to a continuous fill. The primary function for static choropleth maps.

Chart Helpers

Function Purpose
vvn_title() Adds a headline-case title and optional sentence-case subtitle to a ggplot via labs().
vvn_source() Adds a source attribution caption. Required on every published VVN figure.
vvn_note() Adds a footnote-style annotation without a source line.
remove_ticks() Removes axis tick marks. Recommended for bar charts and categorical axes.
remove_axis() Removes an entire axis (text, title, ticks, and line). Use when category labels are self-evident.
scatter_grid() Adds major gridlines on both axes. Shorthand for theme_vvn(grid = "both").
legend_bottom() Moves the legend to the bottom of the chart in horizontal orientation.
remove_legend() Removes the legend entirely. Use when colors are labeled directly on the chart.
vvn_save() Exports a ggplot at 300 dpi, 8 × 5 in, white background. Supports multi-format export in one call (e.g. c("png","pdf","svg")).
vvn_accessibility_check() Computes WCAG 2.1 contrast ratios and reports pass/fail for AA (>= 4.5:1) or AAA (>= 7.0:1) standards.

Tables

Function Purpose
vvn_table() Styles a gt table with VT Maroon headers, alternating gray rows, and an orange accent border. Accepts title, subtitle, and source note.
vvn_table_pvalues() Color-codes a p-value column — maroon for p < 0.001, orange for p < 0.05. Pipe after vvn_table().

Maps

Function Purpose
vvn_map_style() Builds an interactive Leaflet choropleth map with VVN branding: CartoDB muted basemap, VVN palette fill, maroon hover highlight, popup tooltips, and a styled legend.

Shiny UI

Function Purpose
vvn_filter() A branded selectInput dropdown with a maroon label. Use in dashboard sidebars for county, year, or category filters.
vvn_slider() A branded sliderInput with a maroon track. Supports single-value and two-handle range sliders.
vvn_button() A styled actionButton in three variants: primary (maroon fill, for Apply Filters), secondary (orange fill), or outline (maroon border, for Reset).
vvn_kpi_card() Renders a large-number KPI metric card with a label and optional trend arrow. Design in sets of three per row — one maroon, one orange, one navy.

Scaffolding

Function Purpose
create_vvn_dashboard() Generates a complete, runnable Shiny dashboard project in one command: app.R, R/data_prep.R, www/vvn.css, and a data/ folder. Replace placeholder data and deploy.
create_vvn_story() Generates a VVN Insights Quarto project with pre-built narrative sections (Overview, Background, Findings, Conclusion), branded styles, and figures/ + scripts/ folders.
check_vvn_story() Validates a VVN Insights index.qmd before publishing — checks for required sections and confirms theme_vvn() and vvn_source() are used.
