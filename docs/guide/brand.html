WarningColorblind Safety

VT Maroon and VT Orange can appear identical under red-green colorblindness (~8% of men). When both appear on the same chart, add a shape or line-type distinction, or use scale_fill_vvn("accessible").

VT Theme

The VT Theme is the official color system for all standard VVN charts and maps. It is built directly from Virginia Tech’s brand identity.

# Categorical — up to 6 groups in VT brand colors
scale_color_vvn("vt")
scale_fill_vvn("vt")

# Sequential — continuous data (choropleth maps, heatmaps)
scale_fill_vvn_c("maroon_seq")   # light pink → VT Maroon
scale_fill_vvn_c("orange_seq")   # cream → VT Orange
scale_fill_vvn_c("navy_seq")     # sky blue → Navy

# Diverging — data with a meaningful midpoint
scale_fill_vvn_c("vt_div")           # VT Maroon ↔ white ↔ Navy
scale_fill_vvn_c("vt_orange_div")    # Impact Orange ↔ cream ↔ Maroon
vvn_palette("vt_div", n = 11)        # interpolate to any n

# Pick one color by index
vvn_palette("vt", pick = 1)   # #861F41 VT Maroon
vvn_palette("vt", pick = 2)   # #E87722 Impact Orange

Artistic Themes

In addition to VT brand palettes, VVN provides four artistic themes inspired by iconic paintings. These are for editorial, design-forward, or thematic contexts — not default usage.

Each theme has a categorical palette (5 colors) and a diverging variant that interpolates to any number of colors.

# Categorical — assign groups to colors
scale_color_vvn("monet")
scale_fill_vvn("sunflower")

# Pick a single color from a theme
vvn_palette("monet", pick = 3)          # #7FA7A6 deep teal
vvn_palette("academic", pick = 5)       # #F28E2B amber

# Diverging — continuous data with a neutral midpoint
scale_fill_vvn_c("monet_div")
scale_fill_vvn_c("sunflower_div", n = 11)

# Preview any palette
view_vvn_palette("monet")
view_vvn_palette("academic_div", n = 9)

Accessibility Considerations

The VVN color palettes were designed to complement the Virginia Tech brand while supporting effective data visualization. Rather than changing the overall aesthetic, accessibility is achieved through thoughtful application of the existing palettes.

NoteDesign Principle

About 80–90% of the current VVN colors can be retained as-is. Accessibility is met through usage rules and minor adjustments — not a redesign. This preserves the VT brand identity while broadening the audience for every VVN output.

WCAG 2.2 AA Guidelines

To maximize compatibility with WCAG 2.2 AA while preserving the existing visual design:

Rule Guidance
Text Use only dark colors — VT Maroon #861F41, deep navy #1B5299, or charcoal #3D3D3D. Never use light palette colors for body text.
Axes & outlines Use dark colors. Chart axis labels and border lines must meet ≥ 4.5:1 contrast on the background.
Bar / area fills Any palette color is acceptable. Fills are evaluated by contrast relative to their outlines, not the page background.
Background fills Reserve the two lightest colors in each palette (maroon_lt, orange_lt, off-whites) for backgrounds and highlight zones only — not data encoding.
Categorical plots Never rely on color alone. Always combine color with at least one additional encoding: line type, point shape, direct labels, or fill patterns.
All figures Include a descriptive title, a legend or direct labels, a source line, and alt text when embedding in HTML.

Encoding combinations for categorical plots

When color alone may be insufficient (colorblindness, grayscale print), pair it with:

# Line type + color
ggplot(df, aes(x, y, color = group, linetype = group)) +
  geom_line() +
  scale_color_vvn("vt")

# Point shape + color
ggplot(df, aes(x, y, color = group, shape = group)) +
  geom_point(size = 3) +
  scale_color_vvn("accessible")

# Direct labels instead of legend
library(ggrepel)
ggplot(df, aes(x, y, color = group)) +
  geom_line() +
  geom_text_repel(data = . %>% slice_max(x, n = 1, by = group),
                  aes(label = group), nudge_x = 0.3) +
  scale_color_vvn("vt") +
  remove_legend()

TipQuick check

Use vvn_accessibility_check() from vvnthemes to verify any color against any background in R:

vvn_accessibility_check(vvn_palette("vt"), background = "#FFFFFF")
vvn_accessibility_check(c("#FFFFFF"), background = "#861F41")  # white text on maroon

Logo Usage

  • Always use the official VT logo alongside VVN branding
  • Do not alter, recolor, or reposition the logo
  • Minimum clear space: equal to the height of the “V” in “Virginia” on all sides
  • The tagline “Invent the Future®” appears beneath the logo on branded materials
